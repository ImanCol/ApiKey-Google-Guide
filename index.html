A continuación le mostraremos como obtener una Llave Maestra (Api Key), para poder usar esta aplicación.

Ve a la Consola de Google Cloud e inicia sesión con tu cuenta de Google. Debes crear un Proyecto donde puedas alojar el servicio que vamos a instalar.



Si ya tienes un proyecto seleccionalo.



Si no, crea uno, coloca el nombre de tu proyecto.



Aguarda hasta que el proyecto este creado. Después debes ir a Credenciales de Apis y Servicios.



Ve a Biblioteca.


En la barra de búsqueda, escribe Static Map. Selecciona Maps Static API


Habilita el servicio.


Cuando termine de crease el servicio, ve a credenciales.


Crea una Clave API para el Servicio.



Copia el Api Key.



Pega la clave que conseguistes y dale a cualquier opción.



Listo, ya podrás visualizar el Mapa en tiempo real!.
